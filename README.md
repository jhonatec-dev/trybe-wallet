# Functionalities

- Login with email validation
- Main currencies with values updated by the API https://economia.awesomeapi.com.br/
- Record and display in DataGrid
- Real-time graphs of registered expenses


# Technologies used

- React JS
    - redux
    - React Google Chart
    - MUI Components
    - Class Components
